# Portfolio

A server-side rendered portfolio built with Node.js, Express, and EJS.

## Pages

- Home
- Projects
- About
- Contact

## Getting Started

```bash
npm install
npm run dev
```

Then open:

```txt
http://localhost:3000
```

## Personalization

Update the portfolio content in:

```txt
src/data/profile.js
src/data/projects.js
```

## Notes

Do not commit `node_modules/`. Install dependencies from `package.json` instead.
